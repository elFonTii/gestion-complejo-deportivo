Para inicializar la aplicación:

1) Abrir una instancia de powershell
2) Navegar hasta el directorio del proyecto PATH://Desktop/gestion-complejo-deportivo-main/gestion-complejo-deportivo/
3) Ejecutar el comando 'npm run deploy' para desplegar la aplicación en modo de distribución o 'npm run dev' en modo desarrollo.
4) Iniciar una instancia en un navegador e introducir 'localhost' o la ip privada de la máquina.
5) Es importante que no exista otra aplicación escuchando en el puerto 80, ya que eso generaría un conflicto con nuestra aplicación.
6) La aplicación está lista para su utilización.
